# FocusBot

## 中文简介

一款专注于提升工作效率的AI智能助手，支持多种LLM模型的接入和使用，为用户打造专注、高效的智能工作环境。作为一款真正的生产力工具，FocusBot基于先进的大语言模型，帮助用户保持专注，提高工作效率。

## 功能特点

- 支持多种LLM模型接入
- 智能对话管理
- 会话历史记录
- 模型协同调度
- 本地数据存储

## 技术栈

- Electron
- React
- TypeScript
- Ant Design
- Redux Toolkit
- SQLite
- TailwindCSS

## 开发环境要求

- Node.js >= 16.0.0
- npm >= 8.0.0

## 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建应用
npm run build
```

## 项目结构

```
src/
  ├── main/          # Electron 主进程
  ├── renderer/      # React 渲染进程
  ├── shared/        # 共享代码
  └── types/         # TypeScript 类型定义
```
